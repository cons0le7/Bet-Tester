# Bet-Tester
A tool for simulating bets with 50/50 odds to test strategies.


# Depends 
git 

python3


# Installation: 

``` 
git clone https://github.com/cons0le7/Bet-Tester 
cd Bet-Tester
python3 bet.py 
```
